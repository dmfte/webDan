Modo de uso

                
    
                    
  • Crea un prompt con el botón "+ Nuevo Prompt".
    •

    • 
                    
  • Usa "Guardar" para confirmar cambios.
    • 
                    
  • Usa "Copiar" para mandar el texto al portapapeles.
    • 
                    
  • Exporta en JSON para guardarlo o compartirlo.
    • 
                    
  • Gestiona categorías en la pestaña "Categorías".
