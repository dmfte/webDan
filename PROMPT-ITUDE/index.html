Mi Prompt-itud

Biblioteca de prompts lista para copiar

Guarda, agrupa y comparte tus snippets en formato listo para ser usado.

Gestión de Categorías

Edita o elimina categorías. Los prompts huérfanos irán a "Sin categoría".

Modo de uso

  • Crea un prompt con el botón "+ Nuevo Prompt".
  • Abre el acordeón para editar nombre, categoría y texto.
  • Usa "Guardar" para confirmar cambios.
  • Usa "Copiar" para mandar el texto al portapapeles.
  • Exporta en JSON para guardarlo o compartirlo.
  • Gestiona categorías en la pestaña "Categorías".
