Prompt completo

<prompt>
- Read all the tags before you start generating the summary.
- Understand how all tags relate to each other before analyzing the source material.
</prompt>
<task>
- Please provide a concise summary of the main learning points of the attached transcript (or pasted text). Focus on the key ideas, actionable insights, or step-by-step methods the speaker shares.
- Include concepts and definitions provided.
- Avoid filler, repetition; include anecdotes only if they are key to understand a concept or subject.
- Include references to other materials mentioned, such as podcasts, books, online courses, etc.
- The resulting document should enable me to be familiar with the knowledge from transcript without me having to read the transcript.
</task>
<output>
- An HTML document as a downloadable link.
- Use HTML hierarchy to convey the relationship of he material; broad concepts are higher than more detailed descriptions.
- Use HTML structure for nested information; steps as bullet points, details about each step in a nested bullet points list.
- Include a discrete time-stamp on each sub-title, like hh:mm:ss.
- Don't include styles. Only use semantic tags, including, but not limited to: <em>, <strong>, <u>, etc.
</output>
<constrains>
- Don't infer details. Don't research (web) information. Include content only from the attached transcript.
- Don't translate the material. Output information in the same language of the input text.
</constrains>
<important>
- Double check the validity of the download link you provide.
</important>
